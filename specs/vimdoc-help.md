# Vimdoc Help File

## Overview

Create vimdoc help file at `doc/git-worktree.txt` accessible via `:help git-worktree`.

## Requirements

- `doc/git-worktree.txt` exists
- `:help git-worktree` works
- Documents all commands and API
- Includes examples

## Acceptance Criteria

- [ ] `doc/git-worktree.txt` exists

- [ ] `:help git-worktree-config` documents configuration
